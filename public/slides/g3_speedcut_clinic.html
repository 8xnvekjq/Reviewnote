문항 01 [원문]  수학Ⅱ | 다항함수의 적분법 | 2022학년도 9월 모평 14번

Original Question

삼차함수 $f(x) = x^3 - 3x^2$의 극대점 $A$를 지나고 $x$축에 평행한 직선이 이 곡선과 만나는 점 중 $A$가 아닌 점을 $B$라 하자. 곡선 $y=f(x)$와 선분 $AB$로 둘러싸인 도형의 넓이를 구하시오.

문항 01 [원문 해설]  수학Ⅱ | 다항함수의 적분법 | 2022학년도 9월 모평 14번

정석 풀이 (Classic Solution)

연산 4단계

1. 미분하여 극대 위치 탐색:
   $f'(x) = 3x^2 - 6x = 3x(x-2) = 0 \implies x=0$ (극대)

2. 극대점 $A(0,0)$ 및 접선 $y=0$ 설정

3. 교점 $B$ 계산:
   $x^3 - 3x^2 = 0 \implies x^2(x-3) = 0 \implies B(3,0)$

4. 정적분 계산:
   $S = \int_{0}^{3} (0 - f(x)) dx = \left[-\frac{1;}{4}x^4 + x^3\right]_{0}^{3} = \frac{27}{4}$

시간 소요 요인: 다항식 적분 계산량 폭발 및 계산 실수 위험
숏컷 풀이 (Shortcut Skill)

30초 컷 핵심 아이디어

1. 삼차함수 비율 관계 (2:1 법칙)
극대 위치 $x=0$, 극소 위치 $x=2$이므로,
극대 접선과 삼차함수의 교점 $B$의 $x$좌표는 계산 없이 **$x=3$** 도출.

2. 접선 면적 공식 (12 공식)
$$S = \frac{|a|}{12}(\beta - \alpha)^4$$ $$S = \frac{1}{12}(3 - 0)^4 = \frac{81}{12} = \frac{27}{4}$$

비율관계로 점 좌표 즉시 획득 후, 정적분 없이 넓이 공식 대입

문항 01 [구조 변형]  수학Ⅱ | 다항함수의 적분법 | 사선 접선과 넓이

Structure Extension

삼차함수 $f(x) = x^3 - 3x^2 + 2x$와 이 곡선 위의 점 $A(0,0)$에서의 접선 $l$이 있다. 곡선 $y=f(x)$와 접선 $l$로 둘러싸인 도형의 넓이를 구하시오.

문항 02 [원문]  수학Ⅰ | 삼각함수 | 2023학년도 수능 11번

Original Question

방정식 $\cos(\pi x) = \dfrac{1}{4}$의 $0 \le x \le 4$에서의 모든 실근의 합을 구하시오.

문항 02 [원문 해설]  수학Ⅰ | 삼각함수 | 2023학년도 수능 11번

정석 풀이 (Classic Solution)

미지수 매개화 풀이

1. 첫 번째 근을 $x=\alpha$로 정의 ($\cos(\pi\alpha) = \frac{1}{4}$)

2. 삼각함수의 대칭성에 따라 각 실근 표현:
   • 1주기($0 \le x \le 2$): $x_1 = \alpha, \; x_2 = 2-\alpha$
   • 2주기($2 \le x \le 4$): $x_3 = 2+\alpha, \; x_4 = 4-\alpha$

3. 실근 합산:
   $x_1 + x_2 + x_3 + x_4 = \alpha + (2-\alpha) + (2+\alpha) + (4-\alpha) = 8$

시간 소요 요인: 문자가 섞이면서 주기점과 대칭점을 덧셈뺄셈하다 연산 실수 발생
숏컷 풀이 (Shortcut Skill)

20초 컷 핵심 아이디어

1. 선대칭성의 성질 이용
주기 $T = \frac{2\pi}{\pi} = 2$이므로 $0 \le x \le 4$는 **2주기** (근 4개).

2. 대칭축 즉시 합산
• 첫 주기 대칭축: $x = 1 \implies$ 두 근의 합 = $1 \times 2 = 2$
• 둘째 주기 대칭축: $x = 3 \implies$ 두 근의 합 = $3 \times 2 = 6$
• **총합 $= 2 + 6 = \mathbf{8}$**

특수각이 아닐 때는 개별 실근을 잡지 말고 대칭축의 곱으로 다이렉트 해결

문항 02 [구조 변형]  수학Ⅰ | 삼각함수 | 절댓값 대칭형 실근의 합

Structure Extension

방정식 $|\sin(2\pi x)| = \dfrac{1}{3}$의 $0 \le x \le 2$에서의 모든 실근의 합을 구하시오.

문항 03 [원문]  수학Ⅰ | 등차수열 | 2021학년도 6월 모평 12번

Original Question

첫째항이 양수이고 공차가 $d$인 등차수열 $a_n$의 첫째항부터 제$n$항까지의 합 $S_n$이 $S_3 = S_{11}$을 만족시킨다. $S_n$이 최대가 되는 자연수 $n$의 값을 구하시오.

문항 03 [원문 해설]  수학Ⅰ | 등차수열 | 2021학년도 6월 모평 12번

정석 풀이 (Classic Solution)

공식 대입 연립 풀이

1. 합 공식 $S_n = \frac{n(2a+(n-1)d)}{2}$ 대입

2. $S_3 = S_{11} \implies \frac{3(2a+2d)}{2} = \frac{11(2a+10d)}{2} \implies 2a = -13d$

3. $a_1 > 0 \implies d < 0$ 판별

4. $S_n$ 식에 $a = -6.5d$ 대입하여 이차함수 일반식 작성 후 완전제곱식으로 전개하여 꼭짓점 축 구하기

시간 소요 요인: 다항식 전개 과정에서 $a$와 $d$의 비례 상수가 지저분해 연산 유발
숏컷 풀이 (Shortcut Skill)

15초 컷 핵심 아이디어

1. 등차합의 이차함수 모델링
등차수열의 합 $S_n$은 상수항이 없는 **이차함수** $y=S(x)$ 상의 값이다.

2. 이차함수 정중앙 대칭성
$S(3) = S(11)$이므로 대칭축 $x$는 정확히 두 지점의 정중앙인 **$n = \frac{3+11}{2} = 7$** 이다.
$a_1 > 0$이고 수열이 줄어들며 최댓값을 가지므로 축인 **$n=7$**에서 $S_n$은 최대가 된다.

수식 없이 대칭축 공식 $n=\frac{\alpha+\beta}{2}$ 만으로 정답 도출

문항 03 [구조 변형]  수학Ⅰ | 등차수열 | 이산적 대칭 최댓값 조건

Structure Extension

첫째항이 양수이고 공차가 $d$인 등차수열 $a_n$의 첫째항부터 제$n$항까지의 합 $S_n$이 다음 두 조건을 만족시킨다.

(가) $S_5 = S_{16}$

(나) $S_k = S_{k+1}$을 만족하는 자연수 $k$가 존재한다.

이때 $S_n$이 최대가 되는 모든 자연수 $n$의 값의 합을 구하시오.

문항 04 [원문]  미적분 | 수열의 극한 | 2024학년도 수능 28번

Original Question

그림과 같이 한 변의 길이가 $4$인 정사각형 $A_1B_1C_1D_1$에 내접하는 원 $O_1$을 그린다. 원 $O_1$에 외접하고 두 변 $A_1B_1, A_1D_1$에 접하는 가장 큰 원을 $O_2$라 할 때, 동일한 과정을 반복하여 얻은 모든 원 $O_n$들의 넓이의 총합을 구하시오.

원들의 무한 내접 기하 구조

문항 04 [원문 해설]  미적분 | 수열의 극한 | 2024학년도 수능 28번

정석 풀이 (Classic Solution)

좌표평면 연립 풀이

1. 좌표 도입: $A_1(0,4)$, 원 $O_1(2,2)$, $R_1=2$

2. 원 $O_2$의 중심을 $(R_2, R_2)$로 잡음

3. 두 원의 접조건 (중심거리 = 반지름 합):
   $\sqrt{(2-R_2)^2 + (2-R_2)^2} = 2 + R_2$

4. 양변을 제곱하여 2차 방정식 풀기:
   $R_2^2 - 12R_2 + 4 = 0 \implies R_2 = 6 - 4\sqrt{2}$

시간 소요 요인: 양변 제곱 전개 과정 및 수식 계산량이 매우 큼
숏컷 풀이 (Shortcut Skill)

1분 컷 핵심 아이디어

1. 대각선 상의 일직선 정렬 성질
꼭짓점 $A_1$에서 $O_1$ 중심까지의 거리 $2\sqrt{2}$는 기하학적으로 세 성분의 합이다.

$$R_2\sqrt{2} + R_2 + R_1 = 2\sqrt{2}$$ $$R_2(\sqrt{2}+1) + 2 = 2\sqrt{2}$$ $$R_2 = \frac{2(\sqrt{2}-1)}{\sqrt{2}+1} = 2(\sqrt{2}-1)^2$$

• 닮음비: $r = (\sqrt{2}-1)^2 = 3-2\sqrt{2}$

대칭 기하 구조를 이용해 수식 전개 없이 비례식으로 직접 풀이

문항 04 [구조 변형]  미적분 | 부채꼴 내접원의 무한 등비 급수

Structure Extension

그림과 같이 반지름의 길이가 $3$이고 중심각의 크기가 $90^\circ$인 부채꼴 $OP_1Q_1$이 있다. 이 부채꼴에 내접하는 가장 큰 원을 $C_1$이라 하고, 원 $C_1$에 외접하면서 부채꼴의 두 반지름 $OP_1, OQ_1$에 접하는 작은 원을 $C_2$라 하자. 동일한 과정을 반복하여 얻은 모든 원 $C_n$들의 넓이의 총합을 구하시오.

부채꼴 내부 무한 수렴 원 기하 구조

문항 05 [원문]  수학Ⅱ | 함수의 극한 | 2023학년도 9월 모평 20번

Original Question

삼차함수 $f(x)$가 다음 두 조건을 만족시킬 때, $f(2)$의 값을 구하시오.

(가) $\lim\limits_{x \to 0} \dfrac{f(x)}{x} = -3$


(나) $\lim\limits_{x \to 1} \dfrac{f(x)}{x-1} = 2$

문항 05 [원문 해설]  수학Ⅱ | 함수의 극한 | 2023학년도 9월 모평 20번

정석 풀이 (Classic Solution)

다항 일반식 대입 풀이

1. 식 설정: $f(x) = ax^3 + bx^2 + cx + d$

2. (가)에서 $f(0)=0 \implies d=0, \; f'(0)=-3 \implies c=-3$

3. (나)에서 $f(1)=0 \implies a+b-3=0 \implies a+b=3$

4. $f'(1)=2 \implies 3a+2b-3=2 \implies 3a+2b=5$

5. 연립방정식을 연산하여 $a=-1, b=4$ 도출 후 식 완성

시간 소요 요인: 3차 일반 수식을 4개 문자로 연립하면서 쓸데없이 손이 바빠짐
숏컷 풀이 (Shortcut Skill)

40초 컷 핵심 아이디어

1. 인수 정리 기반의 즉시 세팅
$f(0)=0, f(1)=0$ 이므로 식을 다이렉트로 축약한다.
$$f(x) = x(x-1)(ax+b)$$

2. 미분 없는 극한값 정리
• (가) 조건: $x$ 소거 후 $x=0$ 대입 $\implies -b=-3 \implies b=3$
• (나) 조건: $x-1$ 소거 후 $x=1$ 대입 $\implies a+b=2 \implies a=-1$

$$f(x)=x(x-1)(-x+3) \implies f(2)=2(1)(1)=\mathbf{2}$$

연립 대신 인수로 식을 묶고 바로 대입하여 해결

문항 05 [구조 변형]  수학Ⅱ | 함수의 극한 | 미분계수 극점 연립

(가) $\lim\limits_{x \to 1} \dfrac{f(x)}{x-1} = -3$

(나) $f(x)$는 $x=2$에서 극값을 갖는다.
