03. 미분계수 (p.58 ~ 68)

평균변화율과 미분계수의 기하학적 의미, 극한 공식 변형, 미분가능성과 연속성 판별

01. 평균변화율  p.58 ~ 59

Visual Concept
평균변화율 (Average Rate of Change)
구간 $[a, b]$에서 함수 $y=f(x)$의 평균변화율:
$$\frac{\Delta y}{\Delta x} = \frac{f(b) - f(a)}{b - a} = \frac{f(a + \Delta x) - f(a)}{\Delta x}$$

02. 미분계수 (순간변화율)  p.59 ~ 60

Visual Concept
미분계수 (Derivative Coefficient)
함수 $y=f(x)$의 $x=a$에서의 미분계수 $f'(a)$:
$$f'(a) = \lim\limits_{\Delta x \to 0} \frac{f(a + \Delta x) - f(a)}{\Delta x} = \lim\limits_{x \to a} \frac{f(x) - f(a)}{x - a}$$

03. 미분가능성과 연속성  p.61

Visual Concept
미분가능성과 연속성의 관계
함수 $f(x)$가 $x=a$에서 미분가능하면
$f(x)$는 $x=a$에서 반드시 연속이다.

필수예제 01  평균변화율 구하기 (p.62)

Essential Example 01

함수 $f(x) = x^3 - 1$에서 $x$의 값이 $1$에서 $4$까지 변할 때의 평균변화율과 $x=c$에서의 미분계수가 같을 때, 상수 $c$의 값을 구하시오. (단, $1 < c < 4$)

필수예제 02  미세변화량 $\Delta x$ 형태의 미분계수 (p.63)

Essential Example 02

$f'(a) = 1$일 때, 다음 극한값을 구하시오.

(1) $\lim\limits_{h \to 0} \dfrac{f(a + 3h) - f(a)}{h}$


(2) $\lim\limits_{h \to 0} \dfrac{f(a + h^2) - f(a)}{h}$


(3) $\lim\limits_{h \to 0} \dfrac{f(a + 2h) - f(a - h)}{h}$

필수예제 03  $x \to a$ 형태의 미분계수 (p.64)

Essential Example 03

$f(a) = 1, \quad f'(a) = 2$일 때, 다음 극한값을 구하시오.

(1) $\lim\limits_{x \to a} \dfrac{f(x^2) - f(a)}{x - a}$


(2) $\lim\limits_{x \to a} \dfrac{x f(a) - a f(x)}{x - a}$

필수예제 04  관계식이 주어진 경우의 미분계수 (p.65)

Essential Example 04

모든 실수 $x, y$에 대하여 관계식 $$f(x + y) = f(x) + f(y) + 2xy$$ 를 만족시키고 $f'(0) = 1$일 때, $f'(2)$의 값을 구하시오.

필수예제 05  미분계수의 기하학적 의미 (p.66)

Essential Example 05

오른쪽 그림과 같이 곡선 $y=f(x)$ 위의 두 점 $A(a, f(a)), B(b, f(b))$ ($0 < a < b$)에 대하여 다음 세 값의 대소를 비교하시오. $$\frac{f(a)}{a}, \quad \frac{f(b)}{b}, \quad \frac{f(b)-f(a)}{b-a}$$
